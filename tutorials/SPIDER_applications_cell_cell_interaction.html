Using SPIDER for cell-cell interaction analysis

Ruoqiao Chen

2024-09-06

The predicted surface protein abundance values obtained from SPIDER can be used for multiple downstream analysis. Here we provide an example of one downstream application of SPIDER: Cell-cell interaction (CCI) analysis. Typically, CCI analysis are based on the rationale of the co-expression of ligand-receptor pairs in different cell populations, however, due to the lack of surface protein abundance data, most of the current CCI analysis tools such as CellChat can only use RNA expression as a surrogate for surface protein abundance. However, now we can incorporate these typical CCI analysis workflows to directly analyze SPIDER-predicted surface protein abundance data to achieve protein-level CCI analysis.

Install Seurat (version:4), CellChat and load them in R (note that you may need to install them in R in another conda environment different from SPIDER, because CellChat and SPIDER have different requirements for R versions. CellChat and SPIDER do not need to be loaded together for your CCI analysis, so you can have them separately in two different environments):

library(CellChat)
library(Seurat)

Download the demo scRNA-seq data here (this is a subset from the pre-processed data in this paper. For tutorial convenience we only use a subset of it). The data contain the transcriptomic data in single cells from human hepatocellular carcinoma (HCC) patients and healthy controls.

We have already prepared the predicted surface protein abundance files obtained from SPIDER on the scRNA-seq demo data, you can download them here (this is to save your time for running the tutorial. Alternatively, you can also do it by yourself via the SPIDER_predict function).

The protein confidence score files are optional for you to use. Generally, a higher confidence score means that the protein is more likely to be predict accurately. In our paper, we use a threshold of confidence score of >0.85 for selecting highly-confident proteins. Users can also choose some other thresholds depending on their needs. For this tutorial, we will select some highly-confident proteins for the CCI analysis:

highly_confident_proteins_HCC <- unseen_proteins_HCC[confidence_HCC[, 1] > 0.85, ]
highly_confident_proteins_healthy <- unseen_proteins_healthy[confidence_healthy[, 1] > 0.85, ]
shared_highly_confident_proteins <- intersect(rownames(highly_confident_proteins_HCC), rownames(highly_confident_proteins_healthy))

data_combined <- merge(RNA_healthy, RNA_HCC)
data_combined[["ADT"]] <- CreateAssayObject(counts = cbind(rbind(seen_proteins_healthy, unseen_proteins_healthy[shared_highly_confident_proteins, ]), rbind(seen_proteins_HCC, unseen_proteins_HCC[shared_highly_confident_proteins, ])))
data_combined@assays$ADT@data = data_combined@assays$ADT@counts

Having the predicted surface protein abundance data from SPIDER, we can analyze CCIs following a typical CellChat workflow:

#Replace proteins with negative predicted values with zeros, because CellChat cannot input negative data:
data_combined@assays$ADT@counts[is.na(data_combined@assays$ADT@counts)] = 0
data_combined@assays$ADT@counts[data_combined@assays$ADT@counts < 0] = 0
data_combined@assays$ADT@data = data_combined@assays$ADT@counts

#-------------------------------------------------------------
#Cell-cell interaction analysis on imputed ADT:
DefaultAssay(data_combined) = "ADT"

data.combined.Tumor <- data_combined[, data_combined@meta.data$stim == 'Tumor']
data.combined.Control <- data_combined[, data_combined@meta.data$stim == 'Control']

cellchat.Tumor.ADT <- createCellChat(object = data.combined.Tumor, meta = data.combined.Tumor@meta.data, group.by = "predicted.id", assay = 'ADT')
## [1] "Create a CellChat object from a Seurat object"
## Set cell identities for the new CellChat object

cellchat.Control.ADT <- createCellChat(object = data.combined.Control, meta = data.combined.Control@meta.data, group.by = "predicted.id", assay = 'ADT')
## [1] "Create a CellChat object from a Seurat object"
## Set cell identities for the new CellChat object 
## The cell groups used for CellChat analysis are  Inflammatory.Macs, Mature.B.cells, 2.T.cells
for (i in c('cellchat.Tumor.ADT', 'cellchat.Control.ADT')) {
  
  cellchat <- get(i)
  CellChatDB <- CellChatDB.human 
  CellChatDB.use <- CellChatDB
  cellchat@DB <- CellChatDB.use
  cellchat <- subsetData(cellchat) # This step is necessary even if using the whole database
  cellchat <- identifyOverExpressedGenes(cellchat, thresh.pc = 0.1, thresh.fc = 0.25, thresh.p = 0.05, do.fast = FALSE)
  cellchat <- identifyOverExpressedInteractions(cellchat)
  cellchat <- computeCommunProb(cellchat)
  cellchat <- filterCommunication(cellchat, min.cells = 10)
  cellchat <- computeCommunProbPathway(cellchat)
  cellchat <- aggregateNet(cellchat)
  assign(i, cellchat)
  
}

We can show the numbers of interactions among different cell populations under HCC/healthy conditions, respectively:

weight.max <- getMaxWeight(object.list, attribute = c("idents","count"))
par(mfrow = c(1,2), xpd=TRUE)
for (i in 1:length(object.list)) {

  netVisual_circle(tmp, weight.scale = T, label.edge= T, edge.weight.max = weight.max[2], edge.width.max = 4, edge.label.cex = 0.4, vertex.label.cex = 0.4, title.name = paste0("Number of interactions - ", names(object.list)[i]))
}
